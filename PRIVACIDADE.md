# Política de Privacidade — EV Care

Última atualização: 17/06/2026

Esta Política de Privacidade explica como o EV Care coleta, utiliza, armazena e protege informações dos usuários durante o uso do aplicativo.

O EV Care está em fase Beta e pode passar por ajustes de funcionalidades, interface, regras de plano e tratamento de dados.

---

## 1. Quais dados são coletados

O EV Care pode coletar e armazenar os seguintes dados:

- Nome informado pelo usuário
- E-mail de cadastro
- Identificador da conta
- Plano do usuário
- Dados dos veículos cadastrados
- Marca, modelo, versão e dados técnicos do veículo
- Quilometragem atual e histórico de KM
- Registros de recargas
- Custos de recargas
- Locais e observações de recargas, quando informados
- Registros de manutenções
- Serviços personalizados de manutenção
- Feedbacks enviados pelo usuário
- Interesse em recursos Plus, quando informado

---

## 2. Para que os dados são usados

Os dados são usados para:

- Permitir login e identificação da conta
- Separar os dados de cada usuário
- Gerenciar veículos cadastrados
- Calcular estimativas de autonomia, consumo e custo
- Exibir histórico de quilometragem, recargas e manutenções
- Gerar relatórios e exportações
- Exibir alertas de manutenção
- Melhorar o aplicativo com base em feedbacks
- Controlar recursos disponíveis nos planos Free e Plus

---

## 3. Dados de veículo e estimativas

O EV Care utiliza dados informados pelo usuário e dados de referência do catálogo de veículos elétricos.

Essas informações podem variar conforme:

- Versão do veículo
- Ano/modelo
- Condições reais de uso
- Estilo de condução
- Temperatura
- Relevo
- Calibragem dos pneus
- Ciclo de medição usado pelo fabricante

As estimativas do EV Care não substituem o manual do veículo, recomendações do fabricante ou avaliação técnica profissional.

---

## 4. Compartilhamento de dados

O EV Care não vende dados pessoais dos usuários.

Os dados podem ser armazenados e processados em serviços usados para funcionamento do aplicativo, como:

- Supabase, para autenticação e banco de dados
- Streamlit Cloud, para hospedagem da aplicação
- Serviço SMTP configurado para envio de notificações de feedback

Feedbacks enviados pelo usuário podem ser encaminhados por e-mail ao responsável pelo projeto.

---

## 5. Segurança

O EV Care utiliza autenticação e regras de acesso para separar os dados entre usuários.

Cada usuário deve acessar apenas os dados vinculados à própria conta.

Mesmo assim, por estar em fase Beta, o aplicativo pode passar por ajustes de segurança, estrutura e permissões.

---

## 6. Retenção de dados

Os dados podem ser mantidos enquanto a conta do usuário estiver ativa ou enquanto forem necessários para operação, melhoria, segurança e análise do aplicativo.

O usuário pode solicitar exclusão ou revisão de seus dados por meio da aba Feedback.

---

## 7. Direitos do usuário

O usuário pode solicitar:

- Confirmação sobre existência de dados
- Acesso aos dados
- Correção de informações
- Exclusão de dados
- Informações sobre uso dos dados
- Revogação de consentimento, quando aplicável

---

## 8. Dados sensíveis

O usuário não deve inserir senhas, documentos, dados bancários, informações médicas ou informações sensíveis em campos de observação ou feedback.

---

## 9. Alterações nesta política

Esta política pode ser atualizada conforme o EV Care evoluir.

A versão mais recente ficará disponível no repositório do projeto.

---

## 10. Contato

Para dúvidas, solicitações ou feedbacks sobre privacidade, utilize a aba Feedback do EV Care.
